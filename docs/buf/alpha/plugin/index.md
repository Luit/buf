Manage plugins on the Buf Schema Registry.

```
buf alpha plugin [flags]
```

### Flags

```
  -h, --help   help for plugin
```

### Flags inherited from parent commands

```

      --timeout duration    The duration until timing out. (default 2m0s)

```
